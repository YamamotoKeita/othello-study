## Environment
Install the followings.

- Rust 1.66.0
- Node.js 16.14.2
- Tauri-cli 1.2.2

## Setup Project
After cloning, run the following script in the project directory.

```shell
npm install
```

## Run

Use a npm-script named 'tauri' in 'othello-test/package.json' with 'dev' option to run the application.

```shell
npm run tauri dev
```
